## 如何贡献

如果你想为本博客贡献内容或修复问题，请阅读 [贡献指南](CONTRIBUTING.md)。

贡献指南包含：
- 如何创建和提交 Pull Request
- PR 审查标准
- 如何合并 PR 到 master 分支
- 常见问题解答

## 本地开发

```bash
# 安装依赖
npm install

# 本地预览
npm run server

# 构建网站
npm run build

# 清理构建产物
npm run clean
```
